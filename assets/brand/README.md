# Workpad Brand Assets

Workpad uses a floating office-card mark: stacked local cards plus a `W` stroke. The mark should read as a broad office productivity tool, not a role-specific support tool.

## Files

- `workpad-mark.svg`: transparent standalone vector mark
- `workpad-logo.svg`: horizontal logo for light backgrounds
- `workpad-logo-light.svg`: horizontal logo for dark backgrounds
- `workpad-app-icon.svg`: square app icon source
- `workpad-app-icon-256.png`: app icon PNG
- `workpad-app-icon-512.png`: app icon PNG
- `workpad-app-icon.ico`: Windows executable icon
- `workpad-mark-512.png`: standalone mark PNG
- `workpad-logo-960.png`: horizontal logo PNG

## Colors

- Ink: `#161719`
- Teal: `#009688`
- Amber: `#E59B2F`
- Mist: `#DFF2EF`
