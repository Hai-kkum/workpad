# Workpad

오피스 직원을 위한 **완전 로컬·오프라인** 플로팅 카드 메모 도구. 메모장·스티커 메모 여러 개를 펼쳐놓고 쓰는 데스크탑 워크플로우(참조정보 보기 → 메모 작성 → 업무 시스템에 붙여넣기)를, 보안 환경에서 그대로 쓸 수 있게 다시 설계했습니다.

> 포지셔닝: **메모장의 자유도는 그대로, 관리 능력만 더한다.**

**대표 사용 사례 — 고객센터 상담사**: 듀얼 모니터 한쪽이 메모장 창으로 덮이고, 콜마다 메모를 남기고, 고객 정보를 상담 시스템에 붙여넣는 워크플로우에서 출발했습니다. 하지만 반복 텍스트·참조 정보·복사/붙여넣기가 많은 **모든 사무 업무**(공용 오피스 포함)에 똑같이 맞습니다.

## 왜

- 듀얼 모니터 한쪽을 메모장 창 여러 개가 덮고 있는 게 사무실 공통 풍경
- 클라우드 노트(원노트·노션)는 보안 정책으로 불가 + 창을 "펼쳐놓고" 못 씀
- 민감 정보(고객정보 등)가 평문 `.txt`로 PC에 방치 — 보안 사각지대

## 핵심 기능

- **독립 카드 창**: 각 메모가 별도 창. 업무 시스템(상담 시스템 등) 모달·브라우저 옆에 자유롭게 겹쳐 배치. 항상 위 토글, 접기, 자유 크기/위치
- **줄 = 버튼**: 상용구 카드에서 줄을 한 번 클릭하면 복사(아이콘은 호버 시에만, 복사 후 체크). 더블클릭하면 그 줄 편집, 드래그하면 평소처럼 텍스트 선택 — 셋이 충돌하지 않음
- **빈 줄 더블클릭 = 수집**: 카드 맨 아래 빈 줄을 더블클릭하면 클립보드 내용이 플레인 텍스트로 그 자리에 붙고 새 빈 줄이 생김
- **복사 스탬프**: 복사할 때 `[06/12 14:35] 내용`처럼 날짜·시간·사용자ID를 자동으로 붙임(카드별 서식 설정, Shift+클릭은 원문만)
- **자동 저장·복원**: 저장 버튼 없음. 입력 즉시 저장, 크래시·재부팅 후 카드 내용+위치+크기 그대로 복원
- **컨트롤 패널**: 카드 생성, 전체 표시/숨김, 배치 프리셋 저장·복원
- **타임스탬프 메모 자동 파기**: 줄마다 수집 시각이 찍히는 메모(고객센터 콜 메모 등)에서, 지정한 일수가 지난 줄을 자동 삭제

## 보안 설계

- **외부 통신 제로** — 네트워크 코드 없음, CSP로 원격 리소스 차단
- **로컬 암호화 저장** — AES-256-GCM, 데이터 키는 Windows DPAPI(Electron `safeStorage`)로 보호
- **PII 마스킹** — 주민번호·카드번호 등 민감 패턴을 비파괴 오버레이로 가림(원문은 보존, 필요 시 👁로 일시 공개)
- **자동 파기** — 타임스탬프 메모는 TTL 경과 시 자동 삭제
- 오픈소스(MIT) — 코드 공개로 보안 검토 가능, 텔레메트리 없음

## 설치 / 실행

배포본(Windows):

- **설치본** `Workpad-<버전>-setup.exe` — 관리자 권한 불필요, 바탕화면·시작메뉴 바로가기 생성
- **포터블** `Workpad-<버전>-portable.exe` — 무설치 실행
- 메모 데이터는 `%APPDATA%\Workpad`에 저장되어 설치본·포터블이 서로 공유

개발 실행:

```bash
npm install
npm start
```

Windows 10/11 + Node.js 22 권장.

## 로드맵

- 표 카드(셀 클릭 복사, 엑셀 TSV 왕복), 전체 검색 강화
- 선택적 비밀번호 잠금(opt-in, scrypt 데이터 키 암호화)
- 코드 서명 배포(SignPath Foundation)
