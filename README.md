# Workpad (상담 워크패드)

고객센터 상담사를 위한 **완전 로컬, 오프라인** 플로팅 카드 메모 도구입니다. 메모장 여러 개를 펼쳐놓고 쓰는 상담 흐름을 유지하면서, 검색·알림·서식·자동 저장·암호화 저장을 더합니다.

> 포지셔닝: **메모장의 자유도는 그대로, 관리 능력만 더한다.**

## 핵심 기능 (v0.2.6)

- **독립 카드 창**: 노트, 메모, 표가 각각 별도 창으로 열립니다. 항상 위, 접기, 자유 크기/위치, 자동 복원을 지원합니다.
- **노트 줄 카드**: 줄 클릭으로 복사, 더블클릭으로 수정, 드래그로 선택합니다. `전체복사`는 노트 전체를 플레인 텍스트로 복사합니다.
- **파일 업로드**: 패널의 `파일 업로드`로 `.xlsx`, `.csv`, `.tsv`, `.txt`를 노트로 가져옵니다.
- **노트 내 파일 추가**: 노트의 설정 패널(`#`) 안에 있는 `파일 추가`로 기존 노트 뒤에 내용을 이어 붙입니다.
- **검색**: 패널 검색 결과를 더블클릭하면 해당 카드가 열리고 흔들립니다. 메모/노트 안에서는 검색 버튼 또는 `Ctrl+F`로 카드 내부 검색을 합니다.
- **알림**: 카드별 알림 시간을 설정할 수 있습니다. 시간이 되면 카드가 열리고 기본 2회 흔들립니다.
- **서식 툴바**: 하단 툴바에서 배경색, 줄 굵게, 줄 취소선, 체크리스트를 적용합니다. 시각 서식과 말머리 설정은 노트 설정 패널에서 관리합니다.
- **자동 저장·복원**: 입력, 위치, 크기, 표시 상태가 자동 저장되고 재실행 시 복원됩니다.

## 파일 업로드 기준

업로드는 현재 **노트용 1열 데이터**만 지원합니다.

- 여러 행은 가능합니다. 각 행은 노트의 한 줄로 들어갑니다.
- 한 행에 비어 있지 않은 열이 2개 이상이면 가져오기를 중단합니다. 여러 열 표 업로드는 추후 표 기능에서 처리합니다.
- 빈 행은 무시합니다.
- `.xlsx`는 첫 번째로 내용이 있는 시트를 읽습니다.
- 한 셀 안에 줄바꿈이 있으면 첫 줄을 헤더로 보고, 노트 옵션 `상세는 헤더만 보기`로 접어 볼 수 있습니다.
- 노트 내 `파일 추가`는 기존 줄을 지우지 않고 뒤에 이어 붙입니다.
- 암호화 Office 파일, 바이너리 파일을 텍스트 확장자로 바꾼 파일 등은 업로드 전에 차단합니다.

## 서식 범위

- 배경색은 노트, 메모, 표 등 모든 카드에 적용됩니다. 헤더는 같은 계열의 진한 톤으로 자동 지정됩니다.
- 굵게, 취소선, 체크리스트는 노트/할일 줄 단위로 적용됩니다.
- 메모 textarea와 표는 줄 서식을 적용하지 않고 배경색만 적용합니다.
- 화면 서식은 표시 전용입니다. 복사 결과는 항상 플레인 텍스트이며, 선택한 복사 스탬프만 텍스트로 붙습니다.

## 데이터 호환성

- 기존 상용구/콜메모 데이터는 노트로 마이그레이션됩니다.
- 오래된 카드나 백업 복원 데이터에 누락된 필드가 있어도 실행 시 기본값으로 보정합니다.
- 알림 시간, 카드 서식, 줄 데이터는 로컬 저장소에 자동 저장됩니다.

## 보안 설계

- **외부 통신 없음**: 원격 리소스를 쓰지 않는 로컬 앱입니다.
- **로컬 암호화 저장**: AES-256-GCM으로 저장하고, 데이터 키는 Windows DPAPI 또는 비밀번호 잠금으로 보호합니다.
- **개인정보 마스킹**: 주민번호, 카드번호 등 주요 패턴은 화면 표시에서 가릴 수 있습니다.
- **자동 삭제**: 노트 줄은 설정한 보관 기간이 지나면 다음 실행 시 삭제할 수 있습니다.
- **오프라인 배포 가능**: portable/installer 패키징을 지원합니다.

## 보안 문서 업로드 정책

Workpad는 Word/PDF/HTML 문서를 업로드하지 않고, 노트용 `.xlsx`, `.csv`, `.tsv`, `.txt`만 읽습니다. 또한 암호화 Office 패키지나 바이너리 위장 파일은 업로드 전에 차단합니다.

다만 DRM 제품이 사용자 권한으로 파일 내용을 투명하게 복호화해서 일반 앱에 넘겨주는 방식이면, 앱 레벨에서 모든 DRM 문서를 100% 식별할 수 없습니다. 회사 정책상 보안문서의 메모 편입을 확실히 막아야 하는 배포판은 `workpad.config.json`에서 노트 파일 업로드를 끄세요.

```json
{
  "allowDataTransfer": true,
  "allowNoteFileUpload": false
}
```

`allowDataTransfer`를 `false`로 두면 백업/복원과 노트 파일 업로드가 모두 비활성화됩니다.

## 개발 실행

```bash
npm install
npm start
```

Windows 10/11 + Node.js 18+ 환경을 기준으로 합니다.

## 빌드

```bash
npm run pack
npm run dist
```

- `npm run pack`: `dist/win-unpacked` 폴더에 unpacked 앱 생성
- `npm run dist`: portable exe와 installer 생성
